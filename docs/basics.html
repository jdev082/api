Endpoints

/

Displays this documentation.

/fact/specify/:number

Returns a fact specified by the `number` parameter. The `number` parameter is the index of the fact in the `facts` array.

/fact/specify/count/:number

Returns the first `number` number of facts from the `facts` array in a JSON array.

/fact/specify/count/random/:number

Returns `number` number of randomly selected facts from the `facts` array in a JSON array.

/fact/reqs

Returns the number of requests made to the `/fact` endpoint.

/fact

Returns a randomly selected fact from the `facts` array.

/fact/app

Returns a randomly selected fact from the `facts` array specifically for use in applications. Also logs the current time to the console.

/fact/hourly

Returns a fact that is changed every hour.

/fact/hourly/app

Returns a fact that is changed every hour specifically for use in applications. Also logs the current time to the console.
